# Larmor Demo

基于 **Three.js** 的拉莫尔回旋现象三维演示原型项目。

## 1. 项目简介

本项目拟采用 **Three.js + AI 辅助编程 + Vite + GitHub + Vercel** 的技术路线，开发一个可在浏览器中运行的三维演示原型，用于展示带电粒子在磁场中的运动现象。

当前版本定位为 **原型验证（Prototype / MVP）**，首要目标是完成一个**可运行、可交互、可展示**的网页端 3D Demo，为后续扩展 AR 展示、AI 讲解等功能打基础。

## 2. 项目目标

本项目当前阶段的目标包括：

- 搭建 Three.js 网页三维开发环境
- 完成基础三维场景渲染
- 实现粒子运动轨迹的可视化展示
- 提供基础交互功能（如开始、暂停、重置、参数调节）
- 将项目托管到 GitHub，支持版本管理与团队协作
- 部署到 Vercel，生成可访问的在线演示链接

## 3. 当前阶段定位

当前项目处于 **初步开发阶段**，重点是完成以下闭环：

**本地开发 → 原型运行 → GitHub 托管 → 在线部署**

因此，当前版本更强调：

- 原型搭建
- 场景可视化
- 页面可运行
- 演示可交付

而不是一次性完成完整教学系统。

## 4. 技术路线

本项目当前采用如下技术栈：

- **Vite**：前端开发脚手架
- **Three.js**：3D 场景渲染
- **JavaScript**：项目主要开发语言
- **GitHub**：代码托管与版本管理
- **Vercel**：在线部署平台
- **AI 编程工具**：辅助完成代码骨架、界面原型与调试修正

## 5. 计划实现的核心功能

### 5.1 首版原型功能
- 基础 3D 场景搭建
- 坐标轴与视角控制
- 粒子对象展示
- 轨迹线显示
- 磁场方向可视化
- 开始 / 暂停 / 重置按钮
- 基础参数调节面板

### 5.2 后续扩展方向
- 更完整的物理参数控制
- 多粒子展示
- 教学说明面板
- AR 展示扩展
- AI 问答 / 讲解模块
